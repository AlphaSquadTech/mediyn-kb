# How to Manage Admin Sessions

Mediyn gives clinic administrators the ability to oversee and manage sessions at a practice-wide level, including removing sessions when necessary.

## Removing a Single Session

If you need to remove a specific session for a patient:

1. Go to the patient's profile in Mediyn.
2. Find the session you want to remove.
3. Select the administrative **Remove Session** option.
4. Confirm the removal.

## What to Expect

When you remove a session, Mediyn also removes all related records:
- Clinical artifacts and notes
- Uploaded files
- Processing records
- Worksheet drafts and completed worksheets

This ensures no orphaned data remains in the system.

You will see a confirmation showing:
- Which session was removed
- How many related clinical records were removed

## Removing All Sessions for a Patient

In rare cases, you may need to remove all sessions for a specific patient.

1. Go to the patient's profile.
2. Select the administrative option to **Remove All Sessions**.
3. You will be asked to confirm this action before it proceeds.
4. Confirm the bulk removal.

## You'll Need to Provide

- **Confirmation** — You must explicitly confirm the bulk removal. This safeguard prevents accidental data loss.

## What to Expect

After confirming, Mediyn removes every session for that patient along with all related records. You will see a summary showing:
- How many sessions were removed
- How many related clinical records were removed

## Good to Know

- Only clinic administrators can remove sessions through the admin tools.
- Removing a session is permanent. This action cannot be undone. Make sure you truly need to remove the data before confirming.
- The confirmation step for bulk removal is required. You cannot skip it.
- All removed data includes clinical artifacts, uploads, processing records, and worksheets associated with those sessions.
- Consider whether voiding related invoices or superbills is also necessary when removing sessions.
- Use this feature responsibly. It is intended for data cleanup, compliance requirements, or correcting significant errors.
