# How to Manage Sessions

View, find, and update your existing sessions in Mediyn.

## Steps

### Viewing your sessions

1. Open the sessions area in Mediyn.
2. Use the available options to narrow your results:
   - **By therapist** -- See only sessions for a specific therapist
   - **By patient** -- See only sessions for a specific patient
   - **By date range** -- Set a start and end date
   - **By status** -- Choose a specific session stage (see below)
3. Browse through results. Use the next page controls if you have many sessions.

### Viewing a single session

1. Select any session from the list.
2. Review the full details: patient, therapist, times, status, consent status, and any linked information.

### Viewing your daily schedule

Therapists can view their daily queue, which shows all sessions organized by time for a specific date. This gives you a quick overview of your day.

### Rescheduling a session

1. Open the session you need to move.
2. Select the option to reschedule.

**You'll need to provide:**

- New start date and time
- New end date and time

**You can also:**

- Add a reason for the change

3. Confirm the reschedule.

Mediyn cancels the original session and creates a new one at the updated time. The two sessions remain linked so you can see the history.

## What to Expect

When you reschedule, the original session is marked as canceled and a new session is created. The new session references the original, so you always have a clear trail.

Your daily queue groups sessions into organized buckets for easy scanning.

## Good to Know

- Only sessions in the **Scheduled** status can be rescheduled.
- Rescheduling creates a brand-new session. The original is preserved in your records.
- Both therapists and clinic administrators can view and manage sessions.
- Use date range options to quickly find past or upcoming sessions.
