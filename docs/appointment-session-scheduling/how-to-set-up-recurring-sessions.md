# How to Set Up Recurring Sessions

Create a repeating schedule that automatically generates sessions for a patient on a regular basis.

## Steps

### Creating a recurring schedule

1. Open the scheduling area in Mediyn.
2. Select the option to create a recurring schedule.

**You'll need to provide:**

- Therapist
- Patient
- Session type
- Frequency and pattern (for example, weekly, every two weeks)
- Time of day
- Duration in minutes
- Time zone
- Start date
- End date

3. Save the recurring schedule.

Mediyn automatically creates sessions based on your schedule, up to 90 days in advance.

### Viewing your recurring schedules

You can view all recurring schedules and narrow the list by:

- Therapist
- Patient
- Status (Active, Paused, or Cancelled)

### Managing a recurring schedule

You can update a recurring schedule to change its status:

- **Active** -- Sessions are being generated as scheduled
- **Paused** -- No new sessions will be created until you resume
- **Cancelled** -- The recurring schedule is permanently stopped

When canceling a recurring schedule, you can also choose to cancel all future sessions that were already created.

### Generating additional sessions

If you need to extend sessions beyond the current window, you can manually trigger session generation. This is safe to do multiple times -- Mediyn will not create duplicates.

## What to Expect

When you create a recurring schedule, Mediyn immediately generates sessions for up to 90 days. New sessions continue to be created as time passes.

Each generated session is linked to the recurring schedule, so you can always trace where it came from.

The therapist must be assigned to the patient. Mediyn checks this before creating the schedule and again before generating sessions.

## Good to Know

- Pausing a schedule stops new session generation but does not affect sessions already on the calendar.
- Canceling a schedule can optionally cancel all future scheduled sessions at the same time.
- You can generate sessions manually if you need to extend further into the future.
- Each recurring schedule is tied to a specific session type, therapist, and patient.
- If the therapist-patient assignment changes, session generation will stop until the assignment is restored.
