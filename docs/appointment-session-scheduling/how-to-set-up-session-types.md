# How to Set Up Session Types

Define the types of sessions your practice offers so you can use them when scheduling.

## Steps

### Creating a session type

1. Go to your practice settings in Mediyn.
2. Navigate to the session types section.
3. Select the option to create a new session type.

**You'll need to provide:**

- Name (for example, "Individual Therapy" or "Couples Session")
- Procedure code (the billing code for this type of session)
- Duration in minutes
- Rate (the fee for this session type)

**You can also:**

- Choose a color for your calendar display
- Set this as the default session type for your practice

4. Save the new session type.

### Editing a session type

1. Find the session type you want to update.
2. Change any of the fields: name, procedure code, duration, rate, color, or default status.
3. Save your changes.

### Archiving a session type

When you no longer offer a particular session type, you can archive it. Archived types will not appear when scheduling new sessions but remain in your records for past sessions.

## What to Expect

New session types are created with an **Active** status. You can view your session types in two ways:

- **Active** -- Currently in use for scheduling
- **Archived** -- No longer available for new sessions

Your session types appear as options whenever you schedule a session or configure online booking.

## Good to Know

- Only clinic administrators can create, edit, or archive session types.
- Therapists can view the list of available session types but cannot change them.
- Archiving a session type does not affect sessions already scheduled with it.
- You can set one session type as the default. This will be pre-selected when scheduling.
