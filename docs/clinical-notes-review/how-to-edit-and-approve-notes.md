# How to Edit and Approve Notes

Make changes to clinical notes and approve them when they are ready.

## Steps

### Editing a clinical artifact

1. Open the session in Mediyn.
2. Navigate to the clinical notes section.
3. Select the artifact you want to edit (Transcription, Summary, or Key Insights).
4. Make your changes to the content.
5. Save the updated artifact.

Mediyn automatically saves the previous version before applying your changes. This means you always have a complete history of edits.

### Viewing version history

1. Open the artifact you want to review.
2. Select the option to view version history.
3. Browse previous versions of the artifact.

Each version shows the content as it existed before an edit was made. This gives you a full record of changes over time.

### Approving a clinical artifact

1. Open the artifact you want to finalize.
2. Review the content one last time.
3. Select the option to approve.

**You'll need to approve each artifact individually:**

- Transcription
- Summary
- Key Insights

## What to Expect

**After editing:**

The artifact is updated with your changes. The previous content is preserved in the version history. The artifact stays in the **In Review** state until you approve it. Mediyn records who made the edit and when.

**After approving:**

The artifact moves to the **Approved** state. Once all artifacts for a session are approved, the session can move forward to its final stage.

## Good to Know

- You can edit an artifact as many times as needed before approving it. Each edit creates a new version in the history.
- Version history cannot be deleted. This ensures a complete record for compliance.
- Approving an artifact finalizes it. Review carefully before approving.
- Each artifact type is approved separately. You can approve the transcription while still editing the summary.
- The version history shows exactly who made each change and when, providing a clear record.
