# How to Search

Mediyn's global search helps you find patients, sessions, therapists, and documents from one place.

## Steps

1. Click the search bar at the top of any Mediyn screen.
2. Type your search term. You can enter a patient name, therapist name, session detail, or document title.
3. Review the results that appear. Mediyn searches across all categories at once.
4. You can also narrow your results by selecting a specific type, such as patients only or sessions only.

## What to Expect

Mediyn returns a list of matching results grouped by type. Each result shows enough detail to identify the right record. Click any result to open it directly.

## Good to Know

- Search works across your entire practice. You can find any record you have access to.
- Partial matches work. Typing the first few letters of a name will return results.
- You can narrow results to a single type if the full list is too long. For example, choose "Patients" to see only patient matches.
- If you do not find what you are looking for, try a different spelling or a broader term.
