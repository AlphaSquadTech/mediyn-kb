# How to Download Documents

Mediyn provides secure, temporary download links for accessing your stored files.

## Steps

1. Go to the **Documents** section in Mediyn.
2. Find the document you want to download. You can narrow down your search by:
   - **Patient** -- Show documents for a specific patient
   - **Session** -- Show documents linked to a specific session
   - **Category** -- Show only a specific type of document (e.g., consent forms, progress notes)
3. Select the document and choose **Download**.

### What to Expect

- Mediyn generates a secure, temporary download link.
- Use the link to save the file to your device.
- The link expires after a short time. If it expires, simply get a new one.

## Good to Know

- Download links are temporary for security purposes. Each link works for a limited time only.
- You can get a new download link as many times as you need.
- Only documents with a **Ready** status and a **Clean** security scan can be downloaded. Quarantined or failed files are not available.
- You can also access documents through secure messages if they were attached to a conversation.
