# How to Upload Documents

Mediyn makes it easy to upload files securely and organize them by patient, session, and category.

## Steps

1. Go to the **Documents** section in Mediyn.
2. Choose **Upload Document**.

You'll need to provide:
- **Filename** -- The name of the file you are uploading
- **File type** -- The type of file (e.g., PDF, image)
- **Patient** -- The patient this document belongs to
- **Category** -- How to classify this document. Available choices:
  - Intake form
  - Progress notes
  - Consent form
  - HIPAA consent
  - Treatment consent
  - Telehealth consent
  - Insurance
  - Good faith estimate
  - Discharge summary
  - Clinical message attachment
  - Other

You can also:
- **Link to a session** -- Connect the document to a specific session

3. Upload the file.
4. Once the file transfer finishes, confirm the upload is complete.

### What to Expect

- Mediyn starts the upload process and assigns a reference number to your document.
- After the file transfer finishes, you confirm completion. Mediyn then runs an automatic security scan.
- The document moves through these stages: uploading, processing, and finally ready (if the scan passes).

## Completing the Upload

After the file has been transferred, Mediyn asks you to confirm the upload.

### What to Expect

- Mediyn verifies the file and begins the security scan.
- You will see the document's details: filename, file type, size, category, and scan status.
- The scan status starts as **pending** and updates to **clean** once the check is done.
- If a threat is detected, the file is quarantined and will not be available for use.

## Good to Know

- Every file is automatically scanned for security threats. You do not need to do anything extra.
- Documents are not available for download or attachment until the security scan is complete and the file is marked as ready.
- If a file is quarantined, it means the security scan found a potential threat. You may need to re-upload a clean version.
- The file size is recorded for your reference after the upload completes.
