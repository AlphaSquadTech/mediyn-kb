# Files & Documents

Mediyn provides a secure way to upload, manage, and access clinical files and documents for your practice.

## What You Can Do

- Upload documents and link them to patients or sessions
- Categorize files by type (intake forms, consent forms, progress notes, and more)
- Mediyn automatically scans every upload for security threats
- Download documents securely at any time
- Attach documents to secure messages

## Key Concepts

- **Document**: A file you upload to Mediyn, such as a PDF, form, or image. Each document is linked to a patient, a session, or both.
- **Category**: A label that classifies your document. This makes it easy to find specific files later.
- **Security Scan**: Every uploaded file is automatically checked for malware and other threats before it becomes available.
- **Download Link**: A temporary, secure link for accessing a document. Links expire after a short time for security.

## Document Categories

You can classify your documents into these types:

- Intake form
- Progress notes
- Consent form
- HIPAA consent
- Treatment consent
- Telehealth consent
- Insurance
- Good faith estimate
- Discharge summary
- Clinical message attachment
- Other

## Document Stages

Your document moves through these stages after upload:

1. **Uploading** -- The file is being transferred to Mediyn
2. **Processing** -- The file is being scanned for security threats
3. **Ready** -- The file passed all checks and is available for use
4. **Quarantined** -- The file was flagged during the security scan and is not available
5. **Failed** -- The upload did not complete successfully

## Security Scan Status

Each document also has a scan status:

- **Pending** -- The scan is still in progress
- **Clean** -- No threats were detected
- **Infected** -- A threat was found and the file has been quarantined
