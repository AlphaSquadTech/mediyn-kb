# Tips for Files & Documents

## Recommendations

- **Categorize every upload** -- Always assign a category when uploading. This makes it much easier to find documents later, especially as your file library grows.
- **Link documents to sessions when possible** -- Connecting a file to a specific session keeps your clinical records organized and easy to reference.
- **Wait for the security scan to complete** -- Do not try to attach or share a document until its status shows as ready and the scan shows as clean.
- **Use descriptive filenames** -- Name your files clearly (e.g., "Smith_Jane_Intake_Form_2026.pdf") so you can identify them at a glance.
- **Re-upload if a file is quarantined** -- If a file is flagged during the security scan, check the original file on your device. If it is safe, try uploading again. If the problem persists, the file may contain a genuine threat.
- **Download links are temporary by design** -- This protects patient data. Simply get a new link whenever you need to access a file again.

## Common Questions

**Q: What file types can I upload?**
A: Mediyn supports common document types such as PDFs, images, and standard office files. The file type is recorded when you upload.

**Q: How long does the security scan take?**
A: Most scans complete within a few seconds. You will see the status change from pending to clean once the scan finishes.

**Q: What does "quarantined" mean?**
A: A quarantined file was flagged during the security scan for a potential threat. It is not available for download or attachment. You should check the original file and re-upload a clean version if needed.

**Q: Can I organize documents by patient?**
A: Yes. Every document is linked to a patient. You can view all documents for a specific patient from their profile or the documents section.

**Q: Can patients access uploaded documents?**
A: Patients can access documents that are shared with them through secure messages. They do not have direct access to the document library.

**Q: Is there a file size limit?**
A: The file size is recorded after upload. Check with your clinic administrator for any practice-specific size limits.
