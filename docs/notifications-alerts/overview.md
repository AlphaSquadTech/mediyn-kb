# Notifications & Live Updates

Mediyn keeps you informed with in-app notifications and live updates so you never miss an important change.

## What You Can Do

- View your notification feed with all recent alerts and updates
- Filter to see only unread notifications
- Mark notifications as read once you have reviewed them
- Receive live updates when sessions change, new messages arrive, or new documents are created

## Key Concepts

- **Notification feed** — A chronological list of all notifications sent to you. New notifications appear at the top.
- **Unread notifications** — Notifications you have not opened or acknowledged yet. These are highlighted so you can spot them quickly.
- **Live updates** — Real-time alerts that appear automatically in Mediyn as events happen. You do not need to refresh the page.
- **Event types** — Live updates cover session status changes, new notifications, background job completions, and new document creation.
- **Heartbeat** — Mediyn checks the connection every 30 seconds to make sure live updates keep flowing without interruption.
