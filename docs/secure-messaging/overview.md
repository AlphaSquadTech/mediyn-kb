# Secure Messaging

Mediyn provides secure, private messaging between therapists and patients for communication between sessions.

![Messaging](../images/clinic/messaging.png)

## What You Can Do

- Start new conversations with your patients
- Send and receive text messages securely
- Attach documents to messages
- View conversation history and unread message counts
- Patients can message their therapists directly from their own account

## Key Concepts

- **Conversation**: A private, one-on-one messaging thread between a therapist and a patient. Each conversation is linked to both participants.
- **Message**: A single communication within a conversation. Messages can contain text, document attachments, or both.
- **Unread Count**: The number of messages you have not yet read in a conversation. This helps you stay on top of patient communications.
- **Attachments**: Documents you have uploaded to Mediyn can be attached to messages for secure sharing.

## Message Types

Messages in Mediyn can be:

- **Text** -- A standard written message
- **Attachment** -- A message with one or more attached documents
- **Worksheet notification** -- An automatic message generated when a worksheet is shared (created by Mediyn on your behalf)

## Who Can Message

- **Therapists** can start conversations and message any of their assigned patients.
- **Patients** can view their conversations and reply to their therapist.
- **Clinic administrators** can view conversations across the practice.
