# Tips for Secure Messaging

## Recommendations

- **Respond within a reasonable timeframe** -- Patients value timely communication. Check your unread messages regularly between sessions.
- **Use attachments for sharing resources** -- Instead of describing a document, attach it directly to a message so the patient has immediate access.
- **Keep messages concise and clear** -- Short, focused messages are easier for patients to read and act on.
- **Review the unread count daily** -- Make it a habit to check your conversation list for unread messages at the start and end of your day.
- **Use messaging for non-urgent communication** -- Secure messaging is ideal for follow-ups, reminders, and sharing resources. For urgent matters, use your established crisis protocols.
- **Upload documents before composing your message** -- Have your files ready in Mediyn before you start writing so you can attach them smoothly.

## Common Questions

**Q: Is messaging in Mediyn secure?**
A: Yes. All messages are transmitted and stored securely within the Mediyn platform. No message content is sent through email or other unsecured channels.

**Q: Can I see if a patient has read my message?**
A: Yes. Each message includes a read status indicator. You can see the date and time when the recipient viewed it.

**Q: Can patients start conversations?**
A: Conversations are created by therapists. Once a conversation exists, patients can send messages freely within it.

**Q: What kinds of files can I attach?**
A: You can attach any document that has been uploaded to Mediyn and has passed the security scan. See the Files & Documents section for supported file types.

**Q: Can clinic administrators read messages?**
A: Yes. Administrators have access to all conversations across the practice for compliance and oversight.

**Q: What are worksheet notification messages?**
A: When a worksheet is shared with a patient, Mediyn can automatically generate a message in the conversation to let the patient know. These appear as system messages.
