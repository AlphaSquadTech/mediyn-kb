# How to Create Session Packages

Mediyn lets you set up prepaid session bundles for your patients with flexible pricing.

## Steps

1. Open the patient's profile in Mediyn.
2. Go to the **Session Packages** section.
3. Select **Create Package**.
4. Fill in the package details.
5. Confirm and save the package.

## You'll Need to Provide

- **Session type** — The kind of session this package covers
- **Number of sessions** — How many sessions the patient is prepaying for
- **Price** — The total cost for the package
- **Currency** — The currency for the charge

## What to Expect

After you create the package, Mediyn will initiate the payment process. The package starts in a **Pending** stage until the payment is confirmed.

Once payment goes through, the package moves to **Active**. Sessions will be deducted from the package as the patient attends appointments.

## Good to Know

- The patient must have a payment method on file before you create a package.
- The price you set is the total for the entire bundle, not a per-session rate.
- Each package is linked to one session type. If a patient needs packages for different session types, create separate packages.
- You can create multiple active packages for the same patient.
