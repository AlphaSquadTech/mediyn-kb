# Tips for Session Packages

## Recommendations

- **Set clear package sizes** — Common bundle sizes are 4, 8, or 12 sessions. Choose amounts that match your typical treatment plans.

- **Offer a small discount for larger bundles** — Prepaid packages encourage commitment. A modest price break for bigger bundles benefits both you and your patients.

- **Monitor remaining sessions regularly** — Check your patients' package balances so you can suggest renewing before they run out.

- **Create separate packages per session type** — Each package is tied to one session type. If a patient receives both individual and group therapy, set up a package for each.

- **Handle refunds promptly** — If a patient discontinues care, process the refund while usage details are fresh. Mediyn calculates the prorated amount for you.

- **Confirm payment methods first** — Make sure the patient has a saved payment method before creating a package. This prevents packages from getting stuck in the Pending stage.

## Common Questions

**Q: Can a patient have more than one active package?**
A: Yes. A patient can have multiple active packages, even for the same session type.

**Q: What happens when all sessions in a package are used?**
A: The package moves to the Exhausted stage automatically. You can create a new package for the patient to continue.

**Q: Can I change the number of sessions in an existing package?**
A: No. Package details are set when you create them. If you need to adjust, refund the current package and create a new one.

**Q: How is the refund amount calculated?**
A: Mediyn calculates a prorated refund based on the number of unused sessions remaining in the package.

**Q: Who can create and refund packages?**
A: Therapists and clinic administrators can create packages. Only clinic administrators can issue refunds.
