# How to Track Processing Jobs

Monitor the status of your session recordings and transcripts as Mediyn processes them.

## Steps

### Viewing processing jobs

1. Open the session in Mediyn.
2. Navigate to the processing section.
3. Review the list of processing jobs for that session.

Each job shows its current status so you can see exactly where things stand.

### Retrying a failed job

If a processing job did not complete successfully:

1. Find the failed job in the processing list.
2. Select the option to retry.
3. Mediyn will attempt to process the content again.

## What to Expect

Processing jobs run automatically after you submit a transcript or complete a recording upload. Each job handles a specific part of the workflow, such as transcription, summarization, or insight generation.

You can check back at any time to see the current status of each job.

When all processing jobs finish successfully, the session moves forward in its lifecycle and clinical notes become available for your review.

## Good to Know

- Most processing completes within a few minutes.
- If a job fails, you can retry it without re-uploading your content.
- Retrying a job is safe to do multiple times. It will not create duplicate notes.
- If a job continues to fail after retrying, contact Mediyn support for assistance.
- You can also receive live updates about processing status without refreshing the page.
