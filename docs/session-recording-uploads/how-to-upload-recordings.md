# How to Upload Recordings and Transcripts

Submit session content to Mediyn so clinical notes can be generated.

## Steps

### Submitting a transcript and artifacts (recommended)

This is the primary way to get session content into Mediyn.

1. Open the session in Mediyn.
2. Select the option to submit session artifacts.

**You'll need to provide:**

- The de-identified session transcript

**You can also:**

- Include pre-generated clinical artifacts (transcription, summary, and key insights). If you include these, Mediyn will use them directly as draft notes for your review. If you leave them out, Mediyn will generate them automatically from the transcript.
- Include generation details for your records

3. Submit the content.

If you submit the same content more than once, Mediyn recognizes the duplicate and returns the original result. Your data will not be duplicated.

### Uploading an encrypted recording (fallback)

Use this option only when on-device transcription is not available.

1. Open the session in Mediyn.
2. Select the option to upload a file.
3. Choose the upload type:
   - **Encrypted recording** -- For session recordings when transcription was not possible on your device. This is only available when no transcript exists for the session yet.
   - **External document** -- For any other file you want to attach

**You'll need to provide:**

- The file name
- The file type
- The file size

4. Start the upload. Mediyn provides a secure upload location.
5. Upload your file.
6. Complete the upload by confirming the file was sent successfully.

**You can also:**

- Provide a file verification code to confirm the upload is complete

## What to Expect

After submitting a transcript, Mediyn creates processing jobs to generate your clinical notes. You will see a list of artifacts created and any processing jobs that were started.

After completing a file upload, Mediyn begins processing the content automatically.

## Good to Know

- Submitting a transcript with pre-generated artifacts is faster because Mediyn skips the generation step.
- Encrypted recordings are a last resort. Use transcript submission whenever possible.
- Duplicate submissions are handled automatically. It is safe to retry if you are unsure whether your first attempt went through.
- Uploaded files have an expiration window. Complete your upload promptly after starting it.
