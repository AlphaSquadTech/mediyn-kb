# Superbills & Insurance Claims

Mediyn generates HCFA/CMS-1500 superbills so your patients can submit claims for insurance reimbursement.

## What You Can Do

- Create superbills for completed sessions
- Review superbill details including diagnosis and procedure codes
- Finalize superbills and generate downloadable PDFs
- Void superbills that were created in error
- View and search your full superbill history

## Key Concepts

**Superbill** — A standardized receipt (HCFA/CMS-1500 format) that includes the details an insurance company needs to process a reimbursement claim. Patients submit superbills to their insurer.

**Diagnosis codes (ICD-10)** — Standardized codes that describe the patient's diagnosis. These are required on every superbill.

**Procedure code (CPT)** — A code that describes the type of service provided (for example, individual psychotherapy, 45 minutes). Mediyn can pre-fill this based on your session type.

**Superbill stages** — Each superbill moves through these stages:
- **Draft** — The superbill has been created and can still be edited
- **Finalized** — The superbill is locked and a PDF has been generated for download
- **Voided** — The superbill has been canceled and is no longer valid

**Provider details** — Your NPI number, name, and tax identification number appear on the superbill automatically.

**Patient details** — The patient's name, date of birth, address, and insurance information are included on the superbill.
