# Therapist Profiles & Management

Mediyn lets clinic administrators view and manage therapist profiles, credentials, and statuses all in one place.

## What You Can Do

- View a list of all therapists in your clinic
- View detailed profile information for any therapist
- Update therapist names, credentials, and professional details
- Change a therapist's status (active, suspended, or deactivated)
- Set availability templates for scheduling

## Key Concepts

- **Therapist profile** — A record containing the therapist's name, credentials, license information, and preferences.
- **Display name** — The name shown throughout Mediyn. If a preferred name is set, it is used. Otherwise, the first and last name are displayed.
- **Therapist status** — Each therapist has a status that controls their access:
  - **Active** — The therapist can sign in and use Mediyn normally.
  - **Suspended** — The therapist's access is temporarily paused.
  - **Deactivated** — The therapist's account is no longer active.
- **Role** — A therapist can be a **Clinic Admin** (with management permissions) or a **Therapist** (with standard permissions).
- **Availability template** — A recurring weekly schedule that defines when a therapist is available for appointments.
