# Tips for Therapist Profiles & Management

## Recommendations

- **Keep credentials and license info current** — Accurate professional details (NPI, license number, credentials) help with compliance and billing.
- **Use preferred names** — If a therapist goes by a different name, set the preferred name so it displays correctly throughout Mediyn.
- **Set session duration preferences** — Each therapist can have their own default appointment length. This speeds up scheduling.
- **Review therapist statuses regularly** — Suspend or deactivate therapists who are on leave or have left the practice. This keeps your team list clean and secure.
- **Update availability templates when schedules change** — Keeping templates current prevents booking conflicts and missed appointments.

## Common Questions

**Q: Can a therapist update their own profile?**
A: Yes. Therapists can edit their own name, credentials, and preferences from their profile settings.

**Q: What is the difference between Suspended and Deactivated?**
A: Suspended is temporary. Use it when a therapist is on leave. Deactivated is for therapists who have permanently left the practice.

**Q: Does changing a therapist's status cancel their existing appointments?**
A: Changing a status does not automatically cancel appointments. Review and reassign upcoming appointments separately.

**Q: Can I add a new therapist directly instead of sending an invitation?**
A: New therapists join by accepting an email invitation. See [How to Invite Team Members](../getting-started-setup/how-to-invite-team-members.md) for the steps.

**Q: What is the display name?**
A: The display name is automatically set by Mediyn. If a preferred name is provided, it is used. Otherwise, the first and last name are combined.
