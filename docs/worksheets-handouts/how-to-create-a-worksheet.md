# How to Create a Worksheet

Mediyn can recommend the best worksheet template for your session and generate a personalized draft for your patient.

## Getting a Recommendation

1. Open the session you want to create a worksheet for.
2. Choose the **Get Worksheet Recommendations** option.
3. Mediyn analyzes your session notes and any recent clinical assessment data (such as PHQ-9 or GAD-7 scores) to suggest the best templates.

You can also provide:
- **Maximum number of suggestions** -- Limit how many templates Mediyn recommends
- **Include rationale** -- Ask Mediyn to explain why each template was recommended
- **Session note** -- Add extra context to guide the recommendation

### What to Expect

- Mediyn returns a ranked list of recommended templates.
- The top recommendation is highlighted as the default choice.
- If the patient has completed recent assessments, those results are factored in automatically.

## Creating a Draft

1. From the recommendations, select the template you want to use. You can also pick any template from the catalog.
2. Choose **Create Worksheet Draft** for your session.

You'll need to provide:
- **Template** -- The template you selected

You can also:
- **Add a session note** -- Provide extra context to personalize the draft
- **Use manual override** -- Skip the recommendation and choose any template

### What to Expect

- Mediyn generates a draft personalized with your session notes and the patient's clinical data.
- Assessment findings are incorporated using clinically appropriate language. Raw scores are never exposed to the patient.
- The draft includes pre-filled field values you can review and edit.
- If Mediyn detects any content that needs attention, warnings will appear alongside the draft.
- A de-identification report shows what identifying information was flagged or removed.

## Good to Know

- You can create multiple worksheets for the same session if different clinical concerns need to be addressed.
- To edit a draft after creation, update the field values or acknowledge any warnings before moving to the preview step.
- Drafts are visible only to therapists assigned to the patient. Clinic administrators can see all drafts across the practice.
